## 遊び方 (アーカイブ)

- **ルール**: 流れてくる寿司に表示されたローマ字をタイプして取り切ります。
- **コンボ**: 連続して寿司を取ることでスコア倍率がアップします。
- **同時取り**: 複数の寿司を同時に取り抜くと、ボーナススコアが大きく増加します。
